**Full Ownership** — I design the architecture, write the code, deploy to production, and set up observability. I don't wait to be told what to do.

**Product Mindset** — I work directly with business stakeholders, write user stories, and make trade-offs between business value and technical constraints.

**Startup DNA** — Co-founded WavLive: fundraising, team coordination, MVP delivery, KPI tracking. I know what it takes to go from zero to one.

**Engineering Rigor** — DDD, TDD, CI/CD, hexagonal architecture. I build fast without accumulating tech debt.

**Team Building** — Mentored junior developers at OpenClassrooms (100% success rate). Pair programming, code reviews, onboarding — I help teams scale.
